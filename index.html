Reconciliation Logic (Python)


import pandas as pd

# Load reported and adjusted figures
reported = pd.read_csv('data/reported.csv')
adjustments = pd.read_csv('data/adjustments.csv')

# Merge and compute adjusted EBITDA
df = reported.merge(adjustments, on='quarter', how='left')
df['adj_ebitda'] = df['ebitda'] \
  + df['restructuring_costs'] \
  + df['one_time_items']

print(df[['quarter','ebitda','adj_ebitda']].tail())

Financial Formulas

Net Present Value (NPV):

\\[ \mathrm{NPV} = \sum_{t=1}^{T} \frac{CF_t}{(1+r)^t} - C_0 \\]

Capital Asset Pricing Model (CAPM):

\\[ \mathbb{E}[R_i] = R_f + \beta_i \left( \mathbb{E}[R_m] - R_f \right) \\]

Reinforce valuation logic (discounting) and risk-return tradeoff (systematic risk via beta).
